Mais comment peut-on utiliser ce magnifique instrument qu'est le hand spinner ?


Le hand spinner (où fidget spinner pour les Américains) n'est en aucun cas un instrument ! Il s'agit d'une partie intégrante de l'Homme.
Il constitue à la fois une partie du corps (comme vos bras ou vos jambes) mais est aussi une extension de votre être (comme votre smartphone).

Pour vous servir de votre hand spinner, vous devez commencer par le prendre dans votre MAIN comme sur l'image d'archive ci-à gauche :


Veuillez noter que ceci est une façon de tenir votre spinner, à vous de créer votre propre méthode pour vous démarquer des autres personnes !

Mes félicitations ! Vous pouvez dès à présent passer à la seconde étape !

Pour la seconde étape, il va être capital d'utiliser vos DOIGTS à savoir votre index, votre pouce ainsi que votre majeur suivant la position que vous souhaitez adopter. Maintenant vous devez avoir votre hand spinner entre, au choix :

  • l'index et le pouce

  • le majeur et le pouce

  • rien du tout, vous posez donc le hand spinner et si vous faîtes cela alors vous avez compris que vous êtes pas digne !


Voici maintenant une image pour vous montrer comment tenir le hand spinner suivant aussi la position de vos doigt.
Nous baptisons cette position les "Two fingers of God"


Maintenant c'est le moment d'utiliser toutes votre énergie sur cette dernière étape qui va vous demander toute votre attention !
Vous allez avoir le choix entre 2 méthodes suivant le positionnement de vos doigts :

  • Si vous souhaitez utiliser une main de plus, poussez une branche du Hand Spinner avec votre main non occupé vers le bas pour créer l'effet souhaité !

  • Si vous êtes un professionnel de la "toupie à tripoter" (nom défini par Wikipedia), utilisez un doigt pour pousser une branche et vous vanter d'avoir fièrement réussi cette prouesse technique !
